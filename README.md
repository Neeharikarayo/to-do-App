

# ✅ To-Do List Web App

## 📌 Project Description

This is a simple and responsive **To-Do List Web Application** built using **HTML, CSS, and JavaScript**.
The application helps users to **add, manage, and track their daily tasks efficiently**.
It is designed with a clean user interface and smooth functionality to improve productivity and task organization.

---

## 🚀 Features

* ➕ Add new tasks
* ✅ Mark tasks as completed
* 🗑️ Delete tasks
* 📋 View all tasks in a structured list
* 💾 Tasks are stored using **Local Storage** (data will not be lost after refreshing the page)
* 📱 Responsive design for better user experience

---

## 🛠️ Technologies Used

* **HTML** – Structure of the application
* **CSS** – Styling and layout design
* **JavaScript** – Functionality and task management logic

---

## ⚙️ How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Double click on **index.html**
4. The To-Do App will open in your browser
